# Mission Merch Studio App

Clickable MVP for the Mission Merch Studio public pages and first workflow screens.

## Stack

- Vite
- React
- TypeScript
- Tailwind CSS
- React Router

## Commands

```bash
npm install
npm run dev
npm run build
npm run lint
```

## Notes

- Uses sample data in `src/data/sampleData.ts`.
- Checkout is a placeholder; Stripe is intentionally not implemented yet.
- Fulfilment remains manual; no print-on-demand provider integration is included.
